Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A helper class for :ref:`cv::DataType <doxid-da/da2/classcv_1_1_data_type>`.

The class is specialized for each fundamental numerical data type supported by OpenCV. It provides :ref:`DataDepth\<T>::value <doxid-df/daa/classcv_1_1_data_depth_1aed1985524c36b2fc28964a66cccf9038a7cb026110dca902585250642b85975d2>` constant.
