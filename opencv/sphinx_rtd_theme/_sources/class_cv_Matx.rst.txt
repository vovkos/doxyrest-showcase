Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Template class for small matrices whose type and size are known at compilation time.

If you need a more flexible type, use :ref:`Mat <doxid-db/de6/classcv_1_1_mat>`. The elements of the matrix M are accessible using the M(i,j) notation. Most of the common matrix operations (see also :ref:`MatrixExpressions <doxid-d9/ddc/classcv_1_1_mat_expr_1MatrixExpressions>`) are available. To do an operation on :ref:`Matx <doxid-db/d15/classcv_1_1_matx>` that is not implemented, you can easily convert the matrix to :ref:`Mat <doxid-db/de6/classcv_1_1_mat>` and backwards:

.. ref-code-block:: cpp

	Matx33f m(1, 2, 3,
	          4, 5, 6,
	          7, 8, 9);
