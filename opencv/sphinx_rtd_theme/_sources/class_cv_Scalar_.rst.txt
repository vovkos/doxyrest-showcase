Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Template class for a 4-element vector derived from :ref:`Vec <doxid-db/d93/classcv_1_1_vec>`.

Being derived from :ref:`Vec <doxid-db/d93/classcv_1_1_vec>` <_Tp, 4> , :ref:`Scalar\_ <doxid-d7/d13/classcv_1_1_scalar__>` and Scalar can be used just as typical 4-element vectors. In addition, they can be converted to/from :ref:`CvScalar <doxid-de/db0/struct_cv_scalar>`. The type Scalar is widely used in OpenCV to pass pixel values.
