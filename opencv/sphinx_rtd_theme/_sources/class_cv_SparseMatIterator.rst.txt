Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Read-write Sparse Matrix Iterator.

The class is similar to :ref:`cv::SparseMatConstIterator <doxid-db/dc8/classcv_1_1_sparse_mat_const_iterator>`, but can be used for in-place modification of the matrix elements.
