Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The class discriminates between foreground and background pixels by building and maintaining a model of the background.

Any pixel which does not fit this model is then deemed to be foreground. The class implements algorithm described in :ref:`[47] <doxid-d0/de3/citelist_1CITEREF_FGD2003>`.
