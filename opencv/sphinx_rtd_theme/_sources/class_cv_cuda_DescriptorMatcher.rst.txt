Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Abstract base class for matching keypoint descriptors.

It has two groups of match methods: for matching descriptors of an image with another image or with an image set.

-------

	createBFMatcher(int normType = :ref:`cv::NORM_L2<doxid-d2/de8/group__core__array_1ggad12cefbcb5291cf958a85b4b67b6149fa7bacbe84d400336a8f26297d8e80e3a2>`)

Brute-force descriptor matcher.

For each descriptor in the first set, this matcher finds the closest descriptor in the second set by trying each one. This descriptor matcher supports masking permissible matches of descriptor sets.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- normType

		- One of NORM_L1, NORM_L2, NORM_HAMMING. :ref:`L1 <doxid-d0/de0/structcv_1_1_l1>` and :ref:`L2 <doxid-dd/dff/structcv_1_1_l2>` norms are preferable choices for SIFT and SURF descriptors, NORM_HAMMING should be used with :ref:`ORB <doxid-d6/ddf/classcv_1_1cuda_1_1_o_r_b>`, :ref:`BRISK <doxid-dc/d3f/classcv_1_1_b_r_i_s_k>` and BRIEF).

Call this method only after :ref:`DescriptorMatcher::radiusMatchAsync <doxid-db/d2d/classcv_1_1cuda_1_1_descriptor_matcher_1a82d1dfd476176a21739cb99363fa6d0b>` is completed (ie. after synchronization).



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- gpu_matches

		- Matches, returned from :ref:`DescriptorMatcher::radiusMatchAsync <doxid-db/d2d/classcv_1_1cuda_1_1_descriptor_matcher_1a82d1dfd476176a21739cb99363fa6d0b>`.

	*
		- matches

		- Vector of :ref:`DMatch <doxid-d9/db0/classcv_1_1_d_match>` objects.

	*
		- compactResult

		- Parameter used when the mask (or masks) is not empty. If compactResult is false, the matches vector has the same size as queryDescriptors rows. If compactResult is true, the matches vector does not contain matches for fully masked-out query descriptors.

Trains a descriptor matcher.

Trains a descriptor matcher (for example, the flann index). In all methods to match, the method :ref:`train() <doxid-db/d2d/classcv_1_1cuda_1_1_descriptor_matcher_1ab220b434f827962455f430a12c65c074>` is run every time before matching.
