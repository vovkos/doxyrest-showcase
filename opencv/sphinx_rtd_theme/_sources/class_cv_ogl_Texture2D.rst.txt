.. ref-code-block:: cpp

	void
	setAutoRelease(bool flag)

Sets auto release mode.

The lifetime of the OpenGL object is tied to the lifetime of the context. If OpenGL context was bound to a window it could be released at any time (user can close a window). If object's destructor is called after destruction of the context it will cause an error. Thus :ref:`ogl::Texture2D <doxid-d4/d91/classcv_1_1ogl_1_1_texture2_d>` doesn't destroy OpenGL object in destructor by default (all OpenGL resources will be released with OpenGL context). This function can force :ref:`ogl::Texture2D <doxid-d4/d91/classcv_1_1ogl_1_1_texture2_d>` destructor to destroy OpenGL object.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- flag

		- Auto release mode (if true, release will be called in object's destructor).
