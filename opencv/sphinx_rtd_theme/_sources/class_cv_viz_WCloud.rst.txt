Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Clouds.

This 3D :ref:`Widget <doxid-d2/d4e/classcv_1_1viz_1_1_widget>` defines a point cloud. :

In case there are four channels in the cloud, fourth channel is ignored.

.. rubric:: Parameters:

	:widths: 20 80

	*
		- cloud

		- Set of points which can be of type: CV_32FC3, CV_32FC4, CV_64FC3, CV_64FC4.

	*
		- colors

		- Set of colors. It has to be of the same size with cloud.
