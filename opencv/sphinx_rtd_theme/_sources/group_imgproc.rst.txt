	void
	cv::blendLinear(
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` src1,
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` src2,
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` weights1,
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` weights2,
		:ref:`OutputArray<doxid-dc/d84/group__core__basic_1gaad17fda1d0f0d1ee069aebb1df2913c0>` dst
		)

Performs linear blending of two images:

.. math::

	\texttt{dst}(i,j) = \texttt{weights1}(i,j)*\texttt{src1}(i,j) + \texttt{weights2}(i,j)*\texttt{src2}(i,j)



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- src1

		- It has a type of :ref:`CV_8UC(n) <doxid-d1/d1b/group__core__hal__interface_1ga78c5506f62d99edd7e83aba259250394>` or :ref:`CV_32FC(n) <doxid-d1/d1b/group__core__hal__interface_1ga56e67b727727f2f9b73a4b62f0c4b2b5>`, where n is a positive integer.

	*
		- src2

		- It has the same type and size as src1.

	*
		- weights1

		- It has a type of CV_32FC1 and the same size with src1.

	*
		- weights2

		- It has a type of CV_32FC1 and the same size with src1.

	*
		- dst

		- It is created if it does not have the same size and type with src1.
