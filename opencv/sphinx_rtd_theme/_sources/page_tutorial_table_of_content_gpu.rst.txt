GPU-Accelerated Computer Vision (cuda module)
=============================================

Squeeze out every little computation power from your system by using the power of your video card to run the OpenCV algorithms.

* :ref:`Similarity check (PNSR and SSIM) on the GPU <doxid-dd/d3d/tutorial_gpu_basics_similarity>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Bernát Gábor
  
  This will give a good grasp on how to approach coding on the GPU module, once you already know how to handle the other modules. As a test case it will port the similarity methods from the tutorial :ref:`Video Input with OpenCV and similarity measurement <doxid-d5/dc4/tutorial_video_input_psnr_ssim>` to the GPU.

* :ref:`Using a cv::cuda::GpuMat with thrust <doxid-d8/db9/tutorial_gpu_thrust_interop>`
  
  *Compatibility:*>= OpenCV 3.0
  
  This tutorial will show you how to wrap a GpuMat into a thrust iterator in order to be able to use the functions in the thrust library.
