enum cv::FileNode::Type

Overview

type of the file storage node More…

#include <persistence.hpp>

enum Type
{
    NONE      = 0,
    INT       = 1,
    REAL      = 2,
    FLOAT     = REAL,
    STR       = 3,
    STRING    = STR,
    REF       = 4,
    SEQ       = 5,
    MAP       = 6,
    TYPE_MASK = 7,
    FLOW      = 8,
    USER      = 16,
    EMPTY     = 32,
    NAMED     = 64,
};

Detailed Documentation

type of the file storage node

Enum Values

NONE

empty node

INT

an integer

REAL

floating-point number

FLOAT

synonym or REAL

STR

text string in UTF-8 encoding

STRING

synonym for STR

REF

integer of size size_t. Typically used for storing complex dynamic structures where some elements reference the others

SEQ

sequence

MAP

mapping

FLOW

compact representation of a sequence or mapping. Used only by YAML writer

USER

a registered object (e.g. a matrix)

EMPTY

empty structure (sequence or mapping)

NAMED

the node has a name (i.e. it is element of a mapping)
