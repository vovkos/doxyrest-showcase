Detailed Documentation

Generic camera output modes identifier.

Currently, these are supported through the libv4l backend only.

Enum Values

CAP_MODE_BGR

BGR24 (default)

CAP_MODE_RGB

RGB24.

CAP_MODE_GRAY

Y8.
