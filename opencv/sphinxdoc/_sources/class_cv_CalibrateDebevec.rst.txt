Overview
~~~~~~~~

Inverse camera response function is extracted for each brightness value by minimizing an objective function as linear system. Objective function is constructed using pixel values on the same position in all images, extra term is added to make the result smoother. :ref:`More...<details-d6/d56/classcv_1_1_calibrate_debevec>`

Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Inverse camera response function is extracted for each brightness value by minimizing an objective function as linear system. Objective function is constructed using pixel values on the same position in all images, extra term is added to make the result smoother.

For more information see :ref:`[18] <doxid-d0/de3/citelist_1CITEREF_DM97>`.
