Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Template class for 3D points specified by its coordinates ``x``, ``y`` and ``z``.

An instance of the class is interchangeable with the C structure :ref:`CvPoint2D32f <doxid-df/dd9/struct_cv_point2_d32f>`. Similarly to :ref:`Point\_ <doxid-d9/d87/classcv_1_1_point__>`, the coordinates of 3D points can be converted to another type. The vector arithmetic and comparison operations are also supported.

The following :ref:`Point3\_ <doxid-d4/d29/classcv_1_1_point3__>` <> aliases are available:

.. ref-code-block:: cpp

	typedef Point3_<int> Point3i;
	typedef Point3_<float> Point3f;
	typedef Point3_<double> Point3d;
