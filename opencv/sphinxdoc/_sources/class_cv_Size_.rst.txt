Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Template class for specifying the size of an image or rectangle.

The class includes two members called width and height. The structure can be converted to and from the old OpenCV structures :ref:`CvSize <doxid-da/d4d/struct_cv_size>` and :ref:`CvSize2D32f <doxid-de/d91/struct_cv_size2_d32f>`. The same set of arithmetic and comparison operations as for :ref:`Point\_ <doxid-d9/d87/classcv_1_1_point__>` is available.

OpenCV defines the following :ref:`Size\_ <doxid-dd/da0/classcv_1_1_size__>` <> aliases:

.. ref-code-block:: cpp

	typedef Size_<int> Size2i;
	typedef Size2i Size;
	typedef Size_<float> Size2f;
