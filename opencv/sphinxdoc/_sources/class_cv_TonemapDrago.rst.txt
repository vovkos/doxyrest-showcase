Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Adaptive logarithmic mapping is a fast global tonemapping algorithm that scales the image in logarithmic domain.

Since it's a global operator the same function is applied to all the pixels, it is controlled by the bias parameter.

Optional saturation enhancement is possible as described in :ref:`[24] <doxid-d0/de3/citelist_1CITEREF_FL02>`.

For more information see :ref:`[19] <doxid-d0/de3/citelist_1CITEREF_DM03>`.
