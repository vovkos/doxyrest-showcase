.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- image

		- Single-channel 8-bit input image.

	*
		- edges

		- Output edge map. It has the same size and type as image.

	*
		- stream

		- :ref:`Stream <doxid-d1/d04/classcv_1_1cuda_1_1_stream>` for the asynchronous version.

	*

	*

		- Output edge map. It has the same size and type as image.
