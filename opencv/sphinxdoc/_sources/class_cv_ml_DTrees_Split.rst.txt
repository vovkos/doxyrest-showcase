Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The class represents split in a decision tree.

Fields
------

.. index:: pair: variable; c
.. _doxid-df/d74/classcv_1_1ml_1_1_d_trees_1_1_split_1a34ee1a0022d3d297d42aca9302d34fbb:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	float c

The threshold value in case of split on an ordered variable. The rule is:

.. ref-code-block:: cpp

	if var_value < c
	  then next_node <- left
	  else next_node <- right

.. index:: pair: variable; inversed
.. _doxid-df/d74/classcv_1_1ml_1_1_d_trees_1_1_split_1afa6f45d243a85f1ee318b95c9c65b4f4:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	bool inversed

If true, then the inverse split rule is used (i.e. left and right branches are exchanged in the rule expressions below).
