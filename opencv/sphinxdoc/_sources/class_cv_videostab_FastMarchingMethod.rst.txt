.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- mask

		- Image mask. 0 value indicates that the pixel value must be inpainted, 255 indicates that the pixel value is known, other values aren't acceptable.

	*
		- inpaint

		- Inpainting functor that overloads void operator ()(int x, int y).
