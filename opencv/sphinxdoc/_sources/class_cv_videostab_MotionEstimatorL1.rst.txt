Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Describes a global 2D motion estimation method which minimizes :ref:`L1 <doxid-d0/de0/structcv_1_1_l1>` error.

To be able to use this method you must build OpenCV with CLP library support. :
