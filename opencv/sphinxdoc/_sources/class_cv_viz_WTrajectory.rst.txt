	WTrajectory(
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` path,
		int display_mode = :ref:`WTrajectory::PATH<doxid-d3/db6/classcv_1_1viz_1_1_w_trajectory_1a5ad697e1111e7094af72f4a2e3168569a27fc06657f7d3061bd07d6e7192a2006>`,
		double scale = 1.0,
		const :ref:`Color<doxid-df/d5b/classcv_1_1viz_1_1_color>`& color = :ref:`Color::white<doxid-df/d5b/classcv_1_1viz_1_1_color_1a9e57965ee56731f103c1d9e40bd460c1>`()
		)

Constructs a :ref:`WTrajectory <doxid-d3/db6/classcv_1_1viz_1_1_w_trajectory>`.

Frames are not affected. Displays trajectory of the given path as follows:

* PATH : Displays a poly line that represents the path.

* FRAMES : Displays coordinate frames at each pose.

* PATH & FRAMES : Displays both poly line and coordinate frames.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- path

		- List of poses on a trajectory. Takes std::vector<Affine<T>> with T == [float \| double]

	*
		- display_mode

		- Display mode. This can be PATH, FRAMES, and BOTH.

	*
		- scale

		- Scale of the frames. Polyline is not affected.

	*
		- color

		- :ref:`Color <doxid-df/d5b/classcv_1_1viz_1_1_color>` of the polyline that represents path.
