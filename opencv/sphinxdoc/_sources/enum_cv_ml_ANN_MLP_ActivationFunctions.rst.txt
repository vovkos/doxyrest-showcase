	SIGMOID_SYM

Symmetrical sigmoid: :math:`f(x)=\beta*(1-e^{-\alpha x})/(1+e^{-\alpha x}` If you are using the default sigmoid activation function with the default parameter values fparam1=0 and fparam2=0 then the function used is y = 1.7159\*tanh(2/3 \* x), so the output will range from [-1.7159, 1.7159], instead of [0,1].
