Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

SVM kernel type

A comparison of different kernels on the following 2D test case with four classes. Four :ref:`SVM::C_SVC <doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1ab4b93a4c42bbe213ffd9fb3832c6c44fa18157ccaec6a252b901cff6de285d608>` SVMs have been trained (one against rest) with auto_train. Evaluation on three different kernels (:ref:`SVM::CHI2 <doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a66a909b8add6114fde309d24483bcf82>`, :ref:`SVM::INTER <doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a9dec0ceda288deaa617c4c65c88852ae>`, :ref:`SVM::RBF <doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a8e1f51ebeabd14cbd622f0f945831d4c>`). The color depicts the class with max score. Bright means max-score > 0, dark means max-score < 0.

.. ref-code-block:: cpp

	LINEAR

Linear kernel. No mapping is done, linear discrimination (or regression) is done in the original feature space. It is the fastest option. :math:`K(x_i, x_j) = x_i^T x_j`.

.. index:: pair: enumvalue; POLY
.. _doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a5fa32d793cd5f5d0bf64f55bb94a3f2e:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	POLY

Polynomial kernel: :math:`K(x_i, x_j) = (\gamma x_i^T x_j + coef0)^{degree}, \gamma > 0`.

.. index:: pair: enumvalue; RBF
.. _doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a8e1f51ebeabd14cbd622f0f945831d4c:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	RBF

Radial basis function (RBF), a good choice in most cases. :math:`K(x_i, x_j) = e^{-\gamma ||x_i - x_j||^2}, \gamma > 0`.

.. index:: pair: enumvalue; SIGMOID
.. _doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56ac40981025a9b8f3c1cd35cb015aac5cc:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	SIGMOID

Sigmoid kernel: :math:`K(x_i, x_j) = \tanh(\gamma x_i^T x_j + coef0)`.

.. index:: pair: enumvalue; CHI2
.. _doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a66a909b8add6114fde309d24483bcf82:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	CHI2

Exponential Chi2 kernel, similar to the RBF kernel: :math:`K(x_i, x_j) = e^{-\gamma \chi^2(x_i,x_j)}, \chi^2(x_i,x_j) = (x_i-x_j)^2/(x_i+x_j), \gamma > 0`.

.. index:: pair: enumvalue; INTER
.. _doxid-da/d05/classcv_1_1ml_1_1_s_v_m_1aad7f1aaccced3c33bb256640910a0e56a9dec0ceda288deaa617c4c65c88852ae:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	INTER

Histogram intersection kernel. A fast kernel. :math:`K(x_i, x_j) = min(x_i,x_j)`.
