	void
	cv::drawKeypoints(
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` image,

		int flags = :ref:`DrawMatchesFlags::DEFAULT<doxid-d6/d1c/structcv_1_1_draw_matches_flags_1aa2ceb55f7161cc82164e358609453647a48e1d6053ba88b4f14bf1342e6795da6>`
		)

Draws keypoints.

For Python API, flags are modified as cv2.DRAW_MATCHES_FLAGS_DEFAULT, cv2.DRAW_MATCHES_FLAGS_DRAW_RICH_KEYPOINTS, cv2.DRAW_MATCHES_FLAGS_DRAW_OVER_OUTIMG, cv2.DRAW_MATCHES_FLAGS_NOT_DRAW_SINGLE_POINTS



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- image

		- Source image.

	*
		- keypoints

		- Keypoints from the source image.

	*
		- outImage

		- Output image. Its content depends on the flags value defining what is drawn in the output image. See possible flags bit values below.

	*
		- color

		- Color of keypoints.

	*
		- flags

		- Flags setting drawing features. Possible flags bit values are defined by :ref:`DrawMatchesFlags <doxid-d6/d1c/structcv_1_1_draw_matches_flags>`. See details above in drawMatches .

		const :ref:`Scalar<doxid-dc/d84/group__core__basic_1ga599fe92e910c027be274233eccad7beb>`& singlePointColor = :ref:`Scalar::all<doxid-d7/d13/classcv_1_1_scalar___1ac1509a4b8454fe7fe29db069e13a2e6f>`(-1),
		const std::vector<char>& matchesMask = std::vector<char>(),
		int flags = :ref:`DrawMatchesFlags::DEFAULT<doxid-d6/d1c/structcv_1_1_draw_matches_flags_1aa2ceb55f7161cc82164e358609453647a48e1d6053ba88b4f14bf1342e6795da6>`
		)

Draws the found matches of keypoints from two images.

This function draws matches of keypoints from two images in the output image. Match is a line connecting two keypoints (circles). See :ref:`cv::DrawMatchesFlags <doxid-d6/d1c/structcv_1_1_draw_matches_flags>`.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- img1

		- First source image.

	*
		- keypoints1

		- Keypoints from the first source image.

	*
		- img2

		- Second source image.

	*
		- keypoints2

		- Keypoints from the second source image.

	*
		- matches1to2

		- Matches from the first image to the second one, which means that keypoints1[i] has a corresponding point in keypoints2[matches[i]] .

	*
		- outImg

		- Output image. Its content depends on the flags value defining what is drawn in the output image. See possible flags bit values below.

	*
		- matchColor

		- Color of matches (lines and connected keypoints). If matchColor== :ref:`Scalar::all <doxid-d7/d13/classcv_1_1_scalar___1ac1509a4b8454fe7fe29db069e13a2e6f>` (-1) , the color is generated randomly.

	*
		- singlePointColor

		- Color of single keypoints (circles), which means that keypoints do not have the matches. If singlePointColor== :ref:`Scalar::all <doxid-d7/d13/classcv_1_1_scalar___1ac1509a4b8454fe7fe29db069e13a2e6f>` (-1) , the color is generated randomly.

	*
		- matchesMask

		- Mask determining which matches are drawn. If the mask is empty, all matches are drawn.

	*
		- flags

		- Flags setting drawing features. Possible flags bit values are defined by :ref:`DrawMatchesFlags <doxid-d6/d1c/structcv_1_1_draw_matches_flags>`.
