	float
	cv::EMDL1(
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` signature1,
		:ref:`InputArray<doxid-dc/d84/group__core__basic_1ga353a9de602fe76c709e12074a6f362ba>` signature2
		)

Computes the "minimal work" distance between two weighted point configurations base on the papers "EMD-L1: An efficient and Robust Algorithm for comparing histogram-based descriptors", by Haibin Ling and Kazunori Okuda; and "The Earth Mover's Distance is the Mallows Distance: Some Insights from
Statistics", by Elizaveta Levina and Peter Bickel.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- signature1

		- First signature, a single column floating-point matrix. Each row is the value of the histogram in each bin.

	*
		- signature2

		- Second signature of the same format and size as signature1.
