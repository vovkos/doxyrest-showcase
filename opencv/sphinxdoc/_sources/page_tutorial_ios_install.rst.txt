Installation in iOS
===================

.. rubric:: Required Packages

* CMake 2.8.8 or higher

* Xcode 4.2 or higher

.. rubric:: Getting the Cutting-edge OpenCV from Git Repository

Launch GIT client and clone OpenCV repository from `here <http://github.com/opencv/opencv>`__

In MacOS it can be done using the following command in Terminal:

.. ref-code-block:: cpp

	cd ~/<my_working _directory>
	git clone https://github.com/opencv/opencv.git

.. rubric:: Building OpenCV from Source, using CMake and Command Line

#. Make symbolic link for Xcode to let OpenCV build scripts find the compiler, header files etc.
   
   .. ref-code-block:: cpp
   
   	cd /
   	sudo ln -s /Applications/Xcode.app/Contents/Developer Developer

#. Build OpenCV framework:
   
   .. ref-code-block:: cpp
   
   	cd ~/<my_working_directory>
   	python opencv/platforms/ios/build_framework.py ios

If everything's fine, a few minutes later you will get /<my_working_directory>/ios/opencv2.framework. You can add this framework to your Xcode projects.
