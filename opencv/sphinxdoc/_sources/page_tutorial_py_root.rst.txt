OpenCV-Python Tutorials
=======================

* :ref:`Introduction to OpenCV <doxid-da/df6/tutorial_py_table_of_contents_setup>`
  
  Learn how to setup OpenCV-Python on your computer!

* :ref:`Gui Features in OpenCV <doxid-dc/d4d/tutorial_py_table_of_contents_gui>`
  
  Here you will learn how to display and save images and videos, control mouse events and create trackbar.

* :ref:`Core Operations <doxid-d7/d16/tutorial_py_table_of_contents_core>`
  
  In this section you will learn basic operations on image like pixel editing, geometric transformations, code optimization, some mathematical tools etc.

* :ref:`Image Processing in OpenCV <doxid-d2/d96/tutorial_py_table_of_contents_imgproc>`
  
  In this section you will learn different image processing functions inside OpenCV.

* :ref:`Feature Detection and Description <doxid-db/d27/tutorial_py_table_of_contents_feature2d>`
  
  In this section you will learn about feature detectors and descriptors

* :ref:`Video Analysis <doxid-d3/db0/tutorial_py_table_of_contents_video>`
  
  In this section you will learn different techniques to work with videos like object tracking etc.

* :ref:`Camera Calibration and 3D Reconstruction <doxid-d9/db7/tutorial_py_table_of_contents_calib3d>`
  
  In this section we will learn about camera calibration, stereo imaging etc.

* :ref:`Machine Learning <doxid-d6/de2/tutorial_py_table_of_contents_ml>`
  
  In this section you will learn different image processing functions inside OpenCV.

* :ref:`Computational Photography <doxid-d0/d07/tutorial_py_table_of_contents_photo>`
  
  In this section you will learn different computational photography techniques like image denoising etc.

* :ref:`Object Detection <doxid-d9/de5/tutorial_py_table_of_contents_objdetect>`
  
  In this section you will object detection techniques like face detection etc.

* :ref:`OpenCV-Python Bindings <doxid-df/da2/tutorial_py_table_of_contents_bindings>`
  
  In this section, we will see how OpenCV-Python bindings are generated
