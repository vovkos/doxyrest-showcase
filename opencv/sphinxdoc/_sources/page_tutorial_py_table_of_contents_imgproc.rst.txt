Image Processing in OpenCV
==========================

* :ref:`Changing Colorspaces <doxid-df/d9d/tutorial_py_colorspaces>`
  
  Learn to change images between different color spaces. Plus learn to track a colored object in a video.

* :ref:`Geometric Transformations of Images <doxid-da/d6e/tutorial_py_geometric_transformations>`
  
  Learn to apply different geometric transformations to images like rotation, translation etc.

* :ref:`Image Thresholding <doxid-d7/d4d/tutorial_py_thresholding>`
  
  Learn to convert images to binary images using global thresholding, Adaptive thresholding, Otsu's binarization etc

* :ref:`Smoothing Images <doxid-d4/d13/tutorial_py_filtering>`
  
  Learn to blur the images, filter the images with custom kernels etc.

* :ref:`Morphological Transformations <doxid-d9/d61/tutorial_py_morphological_ops>`
  
  Learn about morphological transformations like Erosion, Dilation, Opening, Closing etc

* :ref:`Image Gradients <doxid-d5/d0f/tutorial_py_gradients>`
  
  Learn to find image gradients, edges etc.

* :ref:`Canny Edge Detection <doxid-da/d22/tutorial_py_canny>`
  
  Learn to find edges with Canny Edge Detection

* :ref:`Image Pyramids <doxid-dc/dff/tutorial_py_pyramids>`
  
  Learn about image pyramids and how to use them for image blending

* :ref:`Contours in OpenCV <doxid-d3/d05/tutorial_py_table_of_contents_contours>`
  
  All about Contours in OpenCV

* :ref:`Histograms in OpenCV <doxid-de/db2/tutorial_py_table_of_contents_histograms>`
  
  All about histograms in OpenCV

* :ref:`Image Transforms in OpenCV <doxid-dd/dc4/tutorial_py_table_of_contents_transforms>`
  
  Meet different Image Transforms in OpenCV like Fourier Transform, Cosine Transform etc.

* :ref:`Template Matching <doxid-d4/dc6/tutorial_py_template_matching>`
  
  Learn to search for an object in an image using Template Matching

* :ref:`Hough Line Transform <doxid-d6/d10/tutorial_py_houghlines>`
  
  Learn to detect lines in an image

* :ref:`Hough Circle Transform <doxid-da/d53/tutorial_py_houghcircles>`
  
  Learn to detect circles in an image

* :ref:`Image Segmentation with Watershed Algorithm <doxid-d3/db4/tutorial_py_watershed>`
  
  Learn to segment images with watershed segmentation

* :ref:`Interactive Foreground Extraction using GrabCut Algorithm <doxid-d8/d83/tutorial_py_grabcut>`
  
  Learn to extract foreground with GrabCut algorithm
