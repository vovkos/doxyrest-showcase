Introduction to OpenCV
======================

Here you can read tutorials about how to set up your computer to work with the OpenCV library. Additionally you can find very basic sample source code to introduce you to the world of the OpenCV.

* :ref:`Installation in Linux <doxid-d7/d9f/tutorial_linux_install>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Ana Huamán
  
  We will learn how to setup OpenCV in your computer!

* :ref:`Using OpenCV with gcc and CMake <doxid-db/df5/tutorial_linux_gcc_cmake>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Ana Huamán
  
  We will learn how to compile your first project using gcc and CMake

* :ref:`Using OpenCV with Eclipse (plugin CDT) <doxid-d7/d16/tutorial_linux_eclipse>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Ana Huamán
  
  We will learn how to compile your first project using the Eclipse environment

* :ref:`Installation in Windows <doxid-d3/d52/tutorial_windows_install>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Bernát Gábor
  
  You will learn how to setup OpenCV in your Windows Operating System!

* :ref:`How to build applications with OpenCV inside the "Microsoft Visual Studio" <doxid-d6/d81/tutorial_windows_visual_studio__opencv>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Bernát Gábor
  
  You will learn what steps you need to perform in order to use the OpenCV library inside a new Microsoft Visual Studio project.

* :ref:`Image Watch: viewing in-memory images in the Visual Studio debugger <doxid-d4/d14/tutorial_windows_visual_studio_image_watch>`
  
  *Compatibility:*>= OpenCV 2.4
  
  *Author:* Wolf Kienzle
  
  You will learn how to visualize OpenCV matrices and images within Visual Studio 2012.

* :ref:`Introduction to Java Development <doxid-d9/d52/tutorial_java_dev_intro>`
  
  *Compatibility:*> OpenCV 2.4.4
  
  *Authors:* Eric Christiansen and Andrey Pavlenko
  
  Explains how to build and run a simple desktop Java application using Eclipse, Ant or the Simple Build Tool (SBT).

* :ref:`Using OpenCV Java with Eclipse <doxid-d1/d0a/tutorial_java_eclipse>`
  
  *Compatibility:*> OpenCV 2.4.4
  
  *Author:* Barış Evrim Demiröz
  
  A tutorial on how to use OpenCV Java with Eclipse.

* :ref:`Introduction to OpenCV Development with Clojure <doxid-d7/d1e/tutorial_clojure_dev_intro>`
  
  *Compatibility:*> OpenCV 2.4.4
  
  *Author:* Mimmo Cosenza
  
  A tutorial on how to interactively use OpenCV from the Clojure REPL.

* :ref:`Introduction into Android Development <doxid-d9/d3f/tutorial_android_dev_intro>`
  
  *Compatibility:*> OpenCV 2.4.2
  
  *Author:* Vsevolod Glumov
  
  Not a tutorial, but a guide introducing Android development basics and environment setup

* :ref:`OpenCV4Android SDK <doxid-d9/da6/tutorial__o4_a__s_d_k>`
  
  *Compatibility:*> OpenCV 2.4.2
  
  *Author:* Vsevolod Glumov
  
  OpenCV4Android SDK: general info, installation, running samples

* :ref:`Android Development with OpenCV <doxid-d5/d34/tutorial_dev_with__o_c_v_on__android>`
  
  *Compatibility:*> OpenCV 2.4.3
  
  *Author:* Vsevolod Glumov
  
  Development with OpenCV4Android SDK

* :ref:`Use OpenCL in Android camera preview based CV application <doxid-d7/dbd/tutorial_android_ocl_intro>`
  
  *Compatibility:*>= OpenCV 3.0
  
  *Author:* Andrey Pavlenko
  
  Modify Android camera preview with OpenCL

* :ref:`Installation in iOS <doxid-d5/da3/tutorial_ios_install>`
  
  *Compatibility:*> OpenCV 2.4.2
  
  *Author:* Artem Myagkov, Eduard Feicho
  
  We will learn how to setup OpenCV for using it in iOS!

* :ref:`Cross compilation for ARM based Linux systems <doxid-d0/d76/tutorial_arm_crosscompile_with_cmake>`
  
  *Compatibility:*> OpenCV 2.4.4
  
  *Author:* Alexander Smorkalov
  
  We will learn how to setup OpenCV cross compilation environment for ARM Linux.

* :ref:`Building OpenCV for Tegra with CUDA <doxid-d6/d15/tutorial_building_tegra_cuda>`
  
  *Compatibility:*>= OpenCV 3.1.0
  
  *Author:* Randy J. Ray
  
  This tutorial will help you build OpenCV 3.1.0 for NVIDIA Tegra systems with CUDA 8.0.

* :ref:`Load and Display an Image <doxid-db/deb/tutorial_display_image>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Ana Huamán
  
  We will learn how to display an image using OpenCV

* :ref:`Load, Modify, and Save an Image <doxid-db/d64/tutorial_load_save_image>`
  
  *Compatibility:*> OpenCV 2.0
  
  *Author:* Ana Huamán
  
  We will learn how to save an Image in OpenCV...plus a small conversion to grayscale

* :ref:`Using OpenCV with biicode dependency manager <doxid-d3/d82/tutorial_biicode>`
  
  *Compatibility:*> OpenCV 2.4
  
  *Author:* biicode
  
  We will learn how to setup and use OpenCV in Mac OS X, Linux and Windows

* :ref:`Writing documentation for OpenCV <doxid-d4/db1/tutorial_documentation>`
  
  *Compatibility:*> OpenCV 3.0
  
  *Author:* Maksim Shabunin
  
  This tutorial describes new documenting process and some useful Doxygen features.

* :ref:`Transition guide <doxid-db/dfa/tutorial_transition_guide>`
  
  *Author:* Maksim Shabunin
  
  This document describes some aspects of 2.4 -> 3.0 transition process.
