Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Dummy structure storing DFT/DCT context.

Users can convert this pointer to any type they want. Initialisation and destruction should be made in Init and Free function implementations correspondingly. Example:

.. ref-code-block:: cpp

	int my_hal_dftInit2D(:ref:`cvhalDFT <doxid-dd/d4d/structcvhal_d_f_t>` **context, ...) {
	    *context = static_cast<:ref:`cvhalDFT <doxid-dd/d4d/structcvhal_d_f_t>`*>(new MyFilterData());
	    //... init
	}
	
	int my_hal_dftFree2D(:ref:`cvhalDFT <doxid-dd/d4d/structcvhal_d_f_t>` *context) {
	    MyFilterData *c = static_cast<MyFilterData*>(context);
	    delete c;
	}
