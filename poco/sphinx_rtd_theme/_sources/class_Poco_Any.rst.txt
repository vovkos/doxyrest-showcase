Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

An :ref:`Any <doxid-class_poco_1_1_any>` class represents a general type and is capable of storing any type, supporting type-safe extraction of the internally stored data.

Code taken from the Boost 1.33.1 library. Original copyright by Kevlin Henney. Modified for :ref:`Poco <doxid-namespace_poco>` by Applied Informatics.

Modified for small object optimization support (optionally supported through conditional compilation) by Alex Fabijanic.

Returns the type information of the stored content.

If the :ref:`Any <doxid-class_poco_1_1_any>` is empty typeid(void) is returned. It is recommended to always query an :ref:`Any <doxid-class_poco_1_1_any>` for its type info before trying to extract data via an AnyCast/RefAnyCast.
