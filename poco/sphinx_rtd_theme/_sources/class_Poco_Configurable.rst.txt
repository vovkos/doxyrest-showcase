Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A simple interface that defines :ref:`getProperty() <doxid-class_poco_1_1_configurable_1a565940a99f096c005743a68040ca5e25>` and :ref:`setProperty() <doxid-class_poco_1_1_configurable_1ae31289819bd01f4a1bda0be67144d5bf>` methods.

This interface is implemented by :ref:`Formatter <doxid-class_poco_1_1_formatter>` and :ref:`Channel <doxid-class_poco_1_1_channel>` and is used to configure arbitrary channels and formatters.

A property is basically a name-value pair. For simplicity, both names and values are strings. Every property controls a certain aspect of a :ref:`Formatter <doxid-class_poco_1_1_formatter>` or :ref:`Channel <doxid-class_poco_1_1_channel>`. For example, the :ref:`PatternFormatter <doxid-class_poco_1_1_pattern_formatter>` 's formatting pattern is set via a property.

NOTE: The following property names are use internally by the logging framework and must not be used by channels or formatters:

.. ref-code-block:: cpp

	- class
	- pattern (Channel)
	- formatter (Channel)
