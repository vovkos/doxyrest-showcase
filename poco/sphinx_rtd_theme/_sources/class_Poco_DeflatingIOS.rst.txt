Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`DeflatingOutputStream <doxid-class_poco_1_1_deflating_output_stream>` and :ref:`DeflatingInputStream <doxid-class_poco_1_1_deflating_input_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
