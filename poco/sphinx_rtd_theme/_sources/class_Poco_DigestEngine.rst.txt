Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class is an abstract base class for all classes implementing a message digest algorithm, like :ref:`MD5Engine <doxid-class_poco_1_1_m_d5_engine>` and :ref:`SHA1Engine <doxid-class_poco_1_1_s_h_a1_engine>`.

Call :ref:`update() <doxid-class_poco_1_1_digest_engine_1ac3de0beba3214e7c41b97ceab478f4d1>` repeatedly with data to compute the digest from. When done, call :ref:`digest() <doxid-class_poco_1_1_digest_engine_1acbb5e07fd2f4ae327c3a6897867e07f3>` to obtain the message digest.

	digest() = 0

Finishes the computation of the digest and returns the message digest.

Resets the engine and can thus only be called once for every digest. The returned reference is valid until the next time :ref:`digest() <doxid-class_poco_1_1_digest_engine_1acbb5e07fd2f4ae327c3a6897867e07f3>` is called, or the engine object is destroyed.
