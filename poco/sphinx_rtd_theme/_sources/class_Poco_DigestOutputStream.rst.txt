Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This ostream computes a digest of all the data passing through it, using a :ref:`DigestEngine <doxid-class_poco_1_1_digest_engine>`.

To ensure that all data has been incorporated into the digest, call :ref:`close() <doxid-class_poco_1_1_digest_output_stream_1a9bb59fa923283af82fda95db24c62b91>` or flush() before you obtain the digest from the digest engine.
