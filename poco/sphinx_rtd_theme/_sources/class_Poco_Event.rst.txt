Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

An :ref:`Event <doxid-class_poco_1_1_event>` is a synchronization object that allows one thread to signal one or more other threads that a certain event has happened.

Usually, one thread signals an event, while one or more other threads wait for an event to become signalled.

Construction
------------

	Event(bool autoReset = true)

Creates the event.

If autoReset is true, the event is automatically reset after a :ref:`wait() <doxid-class_poco_1_1_event_1a0da7c72337aab427cdd62307d1bb8769>` successfully returns.
