Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`EventChannel <doxid-class_poco_1_1_event_channel>` fires the messageLogged event for every log message received.

This can be used to hook custom log message processing into the logging framework.
