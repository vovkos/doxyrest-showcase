Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for all :ref:`Formatter <doxid-class_poco_1_1_formatter>` classes.

A formatter basically takes a :ref:`Message <doxid-class_poco_1_1_message>` object and formats it into a string. How the formatting is exactly done is up to the implementation of :ref:`Formatter <doxid-class_poco_1_1_formatter>`. For example, a very simple implementation might simply take the message's Text (see :ref:`Message::getText() <doxid-class_poco_1_1_message_1ad74e81c9260de0217dd74ba5a2237c92>`). A useful implementation should at least take the :ref:`Message <doxid-class_poco_1_1_message>` 's Time, Priority and Text fields and put them into a string.

The :ref:`Formatter <doxid-class_poco_1_1_formatter>` class supports the :ref:`Configurable <doxid-class_poco_1_1_configurable>` interface, so the behaviour of certain formatters is configurable.

Trivial implementations of of :ref:`getProperty() <doxid-class_poco_1_1_formatter_1ade474219b6298aae45df0031a9eccdca>` and :ref:`setProperty() <doxid-class_poco_1_1_formatter_1af080e8c4d794b98c3748b7cfcba16453>` are provided.

Subclasses must at least provide a :ref:`format() <doxid-class_poco_1_1_formatter_1a7dd62f0f22b9d1afb6d49b1a658235ae>` method.
