Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class implements a set using a :ref:`LinearHashTable <doxid-class_poco_1_1_linear_hash_table>`.

A :ref:`HashSet <doxid-class_poco_1_1_hash_set>` can be used just like a std::set.
