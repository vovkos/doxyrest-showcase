.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	double
	avgEntriesPerHash() const

Returns the average number of entries per position in the Hashtable, the higher this value the less efficient performs hashing.

If a large value is returned and getNumberOfZeroPositions also returns a large value, this indicates an inefficient hashing function. If the number of zero entries is low, resizing the :ref:`HashTable <doxid-class_poco_1_1_hash_table>`, should be enough to improve performance
