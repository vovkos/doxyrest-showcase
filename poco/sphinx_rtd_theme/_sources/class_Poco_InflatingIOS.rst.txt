Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`InflatingOutputStream <doxid-class_poco_1_1_inflating_output_stream>` and :ref:`InflatingInputStream <doxid-class_poco_1_1_inflating_input_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
