Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This stream decompresses all data passing through it using zlib's inflate algorithm.

After all data has been written to the stream, :ref:`close() <doxid-class_poco_1_1_inflating_output_stream_1ae4305306ac44110eb9a7ff06813398d4>` must be called to ensure completion of decompression.
