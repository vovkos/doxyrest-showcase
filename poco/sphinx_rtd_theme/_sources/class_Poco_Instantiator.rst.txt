Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A template class for the easy instantiation of instantiators.

For the :ref:`Instantiator <doxid-class_poco_1_1_instantiator>` to work, the class of which instances are to be instantiated must have a no-argument constructor.
