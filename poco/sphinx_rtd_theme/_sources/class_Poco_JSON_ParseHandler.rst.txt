Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

:ref:`ParseHandler <doxid-class_poco_1_1_j_s_o_n_1_1_parse_handler>` is the default handler for the :ref:`JSON <doxid-namespace_poco_1_1_j_s_o_n>` :ref:`Parser <doxid-class_poco_1_1_j_s_o_n_1_1_parser>`.

This handler will construct an :ref:`Object <doxid-class_poco_1_1_j_s_o_n_1_1_object>` or :ref:`Array <doxid-class_poco_1_1_j_s_o_n_1_1_array>` based on the handlers called by the :ref:`Parser <doxid-class_poco_1_1_j_s_o_n_1_1_parser>`.

	ParseHandler(bool preserveObjectOrder = false)

Creates the :ref:`ParseHandler <doxid-class_poco_1_1_j_s_o_n_1_1_parse_handler>`.

If preserveObjectOrder is true, the order of properties inside objects is preserved. Otherwise, items will be sorted by keys.
