.. _details-class_poco_1_1_j_s_o_n_1_1_print_handler:

Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

:ref:`PrintHandler <doxid-class_poco_1_1_j_s_o_n_1_1_print_handler>` formats and prints the :ref:`JSON <doxid-namespace_poco_1_1_j_s_o_n>` object to either user-provided std::ostream or standard output.

If indent is zero, the output is a condensed :ref:`JSON <doxid-namespace_poco_1_1_j_s_o_n>` string, otherwise, the proper indentation is applied to elements.
