Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The implementation of :ref:`LogFile <doxid-class_poco_1_1_log_file>` for non-Windows platforms.

The implementation of :ref:`LogFile <doxid-class_poco_1_1_log_file>` for Windows.

The implementation of :ref:`LogFile <doxid-class_poco_1_1_log_file>` for OpenVMS.

The native filesystem APIs are used for total control over locking behavior.

The C Standard Library functions for file I/O are used with OpenVMS-specific extensions to control sharing and locking behavior.
