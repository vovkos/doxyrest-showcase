Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class implements a streambuf interface to a :ref:`Logger <doxid-class_poco_1_1_logger>`.

The streambuf appends all characters written to it to a string. As soon as a CR or LF (std::endl) is written, the string is sent to the :ref:`Logger <doxid-class_poco_1_1_logger>`, with the set priority.
