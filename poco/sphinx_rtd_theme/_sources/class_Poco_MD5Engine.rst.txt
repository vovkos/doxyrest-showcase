	digest()

Finishes the computation of the digest and returns the message digest.

Resets the engine and can thus only be called once for every digest. The returned reference is valid until the next time :ref:`digest() <doxid-class_poco_1_1_m_d5_engine_1a4979f57b642296e1e5ea030e4a448fae>` is called, or the engine object is destroyed.
