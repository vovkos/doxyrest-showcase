Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A :ref:`Manifest <doxid-class_poco_1_1_manifest>` maintains a list of all classes contained in a dynamically loadable class library.

Internally, the information is held in a map. An iterator is provided to iterate over all the classes in a :ref:`Manifest <doxid-class_poco_1_1_manifest>`.
