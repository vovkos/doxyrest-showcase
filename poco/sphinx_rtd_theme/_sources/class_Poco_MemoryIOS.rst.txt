Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`MemoryInputStream <doxid-class_poco_1_1_memory_input_stream>` and :ref:`MemoryOutputStream <doxid-class_poco_1_1_memory_output_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
