Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A SingletonMetaObject disables the :ref:`create() <doxid-class_poco_1_1_meta_singleton_1a19361db9fa33f075b53cf2e435d6b887>` method and instead offers an :ref:`instance() <doxid-class_poco_1_1_meta_singleton_1a9f795e716bceeedda140e8077e1f17dd>` method to access the single instance of its class.

	bool
	canCreate() const

Returns false if the class is a singleton.
