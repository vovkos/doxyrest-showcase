Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for all password providers.

An instance of a subclass of this class can be registered with the :ref:`FTPStreamFactory <doxid-class_poco_1_1_net_1_1_f_t_p_stream_factory>` to provide a password
