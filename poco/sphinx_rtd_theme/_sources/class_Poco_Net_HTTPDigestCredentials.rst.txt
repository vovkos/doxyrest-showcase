Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This is a utility class for working with HTTP Digest Authentication in :ref:`HTTPRequest <doxid-class_poco_1_1_net_1_1_h_t_t_p_request>` objects.

Note: currently, no qop or qop=auth is supported only.
