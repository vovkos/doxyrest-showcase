Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class encapsulates an HTTP request message.

In addition to the properties common to all HTTP messages, a HTTP request has a method (e.g. GET, HEAD, POST, etc.) and a request :ref:`URI <doxid-class_poco_1_1_u_r_i>`.
