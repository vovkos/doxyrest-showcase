Creates a new request handler for the given HTTP request.

The method should inspect the given :ref:`HTTPServerRequest <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_request>` object (e.g., method and :ref:`URI <doxid-class_poco_1_1_u_r_i>`) and create an appropriate :ref:`HTTPRequestHandler <doxid-class_poco_1_1_net_1_1_h_t_t_p_request_handler>` object to handle the request.
