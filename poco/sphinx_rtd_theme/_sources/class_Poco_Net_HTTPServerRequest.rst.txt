Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This abstract subclass of :ref:`HTTPRequest <doxid-class_poco_1_1_net_1_1_h_t_t_p_request>` is used for representing server-side HTTP requests.

A :ref:`HTTPServerRequest <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_request>` is passed to the handleRequest() method of :ref:`HTTPRequestHandler <doxid-class_poco_1_1_net_1_1_h_t_t_p_request_handler>`.
