Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This ostream encodes all data written to it in quoted-printable encoding (see RFC 2045) and forwards it to a connected ostream.

Always call :ref:`close() <doxid-class_poco_1_1_net_1_1_quoted_printable_encoder_i_o_s_1ac994ff2c79fd175ada9b6833a5a53f49>` when done writing data, to ensure proper completion of the encoding operation.
