Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`SocketStream <doxid-class_poco_1_1_net_1_1_socket_stream>`, :ref:`SocketInputStream <doxid-class_poco_1_1_net_1_1_socket_input_stream>` and :ref:`SocketOutputStream <doxid-class_poco_1_1_net_1_1_socket_output_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
