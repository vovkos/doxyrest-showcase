Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`NullChannel <doxid-class_poco_1_1_null_channel>` is the /dev/null of Channels.

A :ref:`NullChannel <doxid-class_poco_1_1_null_channel>` discards all information sent to it. Furthermore, its :ref:`setProperty() <doxid-class_poco_1_1_null_channel_1acbda8d2663366e0e13a36f7d2ff51060>` method ignores all properties, so it the :ref:`NullChannel <doxid-class_poco_1_1_null_channel>` has the nice feature that it can stand in for any other channel class in a logging configuration.
