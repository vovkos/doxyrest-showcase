Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`NullInputStream <doxid-class_poco_1_1_null_input_stream>` and :ref:`NullOutputStream <doxid-class_poco_1_1_null_output_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
