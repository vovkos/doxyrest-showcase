Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A :ref:`NullMutex <doxid-class_poco_1_1_null_mutex>` is an empty mutex implementation which performs no locking at all.

Useful in policy driven design where the type of mutex used can be now a template parameter allowing the user to switch between thread-safe and not thread-safe depending on his need Works with the :ref:`ScopedLock <doxid-class_poco_1_1_scoped_lock>` class
