Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

:ref:`Nullable <doxid-class_poco_1_1_nullable>` is a simple wrapper class for value types that allows objects or native type variables to have "null" value.

The class is useful for passing parameters to functions when parameters are optional and no default values should be used or when a non-assigned state is needed, such as in e.g. fetching null values from database.

A :ref:`Nullable <doxid-class_poco_1_1_nullable>` can be default constructed. In this case, the :ref:`Nullable <doxid-class_poco_1_1_nullable>` will have a Null value and :ref:`isNull() <doxid-class_poco_1_1_nullable_1a2164d24e6d7d57a782d2bc641a4bdbdd>` will return true. Calling :ref:`value() <doxid-class_poco_1_1_nullable_1a22045b5bf3691fedab7706dcf0244e61>` (without default value) on a Null object will throw a NullValueException.

A :ref:`Nullable <doxid-class_poco_1_1_nullable>` can also be constructed from a value. It is possible to assign a value to a :ref:`Nullable <doxid-class_poco_1_1_nullable>`, and to reset a :ref:`Nullable <doxid-class_poco_1_1_nullable>` to contain a Null value by calling :ref:`clear() <doxid-class_poco_1_1_nullable_1a50cf9981c73db555887e293f6cde5c96>`.

For use with :ref:`Nullable <doxid-class_poco_1_1_nullable>`, the value type should support default construction.
