Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`NumberFormatter <doxid-class_poco_1_1_number_formatter>` class provides static methods for formatting numeric values into strings.

There are two kind of static member functions:

.. ref-code-block:: cpp

	* format* functions return a std::string containing
	  the formatted value.
	* append* functions append the formatted value to
	  an existing string.
