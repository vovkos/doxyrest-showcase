Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`PipeInputStream <doxid-class_poco_1_1_pipe_input_stream>` and :ref:`PipeOutputStream <doxid-class_poco_1_1_pipe_output_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
