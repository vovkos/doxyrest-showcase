Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

An input stream for reading from a :ref:`Pipe <doxid-class_poco_1_1_pipe>`.

Using formatted input from a :ref:`PipeInputStream <doxid-class_poco_1_1_pipe_input_stream>` is not recommended, due to the read-ahead behavior of istream with formatted reads.
