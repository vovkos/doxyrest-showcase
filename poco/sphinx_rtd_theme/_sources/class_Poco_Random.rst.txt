Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A better random number generator.

:ref:`Random <doxid-class_poco_1_1_random>` implements a pseudo random number generator (PRNG). The PRNG is a nonlinear additive feedback random number generator using 256 bytes of state information and a period of up to 2^69.
