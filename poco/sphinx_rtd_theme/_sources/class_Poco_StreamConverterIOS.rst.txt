Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`InputStreamConverter <doxid-class_poco_1_1_input_stream_converter>` and :ref:`OutputStreamConverter <doxid-class_poco_1_1_output_stream_converter>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
