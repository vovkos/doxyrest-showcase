Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`TeeInputStream <doxid-class_poco_1_1_tee_input_stream>` and :ref:`TeeOutputStream <doxid-class_poco_1_1_tee_output_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
