Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class manages the local storage for each thread.

Never use this class directly, always use the :ref:`ThreadLocal <doxid-class_poco_1_1_thread_local>` template for managing thread local storage.
