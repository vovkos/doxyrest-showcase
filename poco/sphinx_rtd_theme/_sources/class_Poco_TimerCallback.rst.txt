Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This template class implements an adapter that sits between a :ref:`Timer <doxid-class_poco_1_1_timer>` and an object's method invoked by the timer.

It is quite similar in concept to the :ref:`RunnableAdapter <doxid-class_poco_1_1_runnable_adapter>`, but provides some :ref:`Timer <doxid-class_poco_1_1_timer>` specific additional methods. See the :ref:`Timer <doxid-class_poco_1_1_timer>` class for information on how to use this template class.
