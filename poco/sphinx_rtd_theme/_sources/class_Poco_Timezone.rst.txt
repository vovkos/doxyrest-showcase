	bool
	isDst(:ref:`const<doxid-zconf_8h_1a2c212835823e3c54a8ab6d95c652660e>` :ref:`Timestamp<doxid-class_poco_1_1_timestamp>`& timestamp)

Returns true if daylight saving time is in effect for the given time.

Depending on the operating system platform this might only work reliably for certain date ranges, as the C library's localtime() function is used.

	static
	int
	tzd()

Returns the time zone differential for the current timezone.

The timezone differential is computed as :ref:`utcOffset() <doxid-class_poco_1_1_timezone_1a3eae063a366df8a8e5787150d879875e>` + :ref:`dst() <doxid-class_poco_1_1_timezone_1a0da2b1bbfdd43142c974e501124674b8>` and is expressed in seconds.
