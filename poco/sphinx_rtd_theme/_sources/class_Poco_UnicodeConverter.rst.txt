Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A convenience class that converts strings from UTF-8 encoded std::strings to UTF-16 or UTF-32 encoded std::wstrings and vice-versa.

This class is mainly used for working with the :ref:`Unicode <doxid-class_poco_1_1_unicode>` Windows APIs and probably won't be of much use anywhere else ???
