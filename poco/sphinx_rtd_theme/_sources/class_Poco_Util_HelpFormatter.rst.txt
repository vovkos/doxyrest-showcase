Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class formats a help message from an :ref:`OptionSet <doxid-class_poco_1_1_util_1_1_option_set>`.

Construction
------------

.. index:: pair: function; HelpFormatter
.. _doxid-class_poco_1_1_util_1_1_help_formatter_1af72fe8854c854cb7409a4360b080f210:

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	HelpFormatter(:ref:`const<doxid-zconf_8h_1a2c212835823e3c54a8ab6d95c652660e>` :ref:`OptionSet<doxid-class_poco_1_1_util_1_1_option_set>`& options)

Creates the :ref:`HelpFormatter <doxid-class_poco_1_1_util_1_1_help_formatter>`, using the given options.

The :ref:`HelpFormatter <doxid-class_poco_1_1_util_1_1_help_formatter>` just stores a reference to the given :ref:`OptionSet <doxid-class_poco_1_1_util_1_1_option_set>`, so the :ref:`OptionSet <doxid-class_poco_1_1_util_1_1_option_set>` must not be destroyed during the lifetime of the :ref:`HelpFormatter <doxid-class_poco_1_1_util_1_1_help_formatter>`.

	void
	setUnixStyle(bool flag)

Enables Unix-style options.

Both short and long option names are printed if Unix-style is set. Otherwise, only long option names are printed.

After calling :ref:`setUnixStyle() <doxid-class_poco_1_1_util_1_1_help_formatter_1a7d2a70183916043b16c143efb4232eed>`, :ref:`setAutoIndent() <doxid-class_poco_1_1_util_1_1_help_formatter_1ab82a49ffc63a47ffa08c4a3adfc5984f>` should be called as well to ensure proper help text formatting.
