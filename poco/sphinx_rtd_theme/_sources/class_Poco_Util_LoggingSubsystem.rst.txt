Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`LoggingSubsystem <doxid-class_poco_1_1_util_1_1_logging_subsystem>` class initializes the logging framework using the :ref:`LoggingConfigurator <doxid-class_poco_1_1_util_1_1_logging_configurator>`.

It also sets the :ref:`Application <doxid-class_poco_1_1_util_1_1_application>` 's logger to the logger specified by the "application.logger" property, or to "Application" if the property is not specified.
