Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

:ref:`Validator <doxid-class_poco_1_1_util_1_1_validator>` specifies the interface for option validators.

:ref:`Option <doxid-class_poco_1_1_util_1_1_option>` validators provide a simple way for the automatic validation of command line argument values.
