Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A dummy class with value-type semantics, mostly useful as a template argument.

This class is typically used together with :ref:`ActiveMethod <doxid-class_poco_1_1_active_method>`, if no argument or return value is needed.
