Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This is a convenience class that combines a :ref:`DOMBuilder <doxid-class_poco_1_1_x_m_l_1_1_d_o_m_builder>` with a :ref:`SAXParser <doxid-class_poco_1_1_x_m_l_1_1_s_a_x_parser>`, with the optional support of a :ref:`WhitespaceFilter <doxid-class_poco_1_1_x_m_l_1_1_whitespace_filter>`.

	DOMParser(unsigned long namePoolSize)

Creates a new :ref:`DOMParser <doxid-class_poco_1_1_x_m_l_1_1_d_o_m_parser>`, using the given :ref:`NamePool <doxid-class_poco_1_1_x_m_l_1_1_name_pool>` size.

The given namePoolSize should be a suitable prime number, e.g. 251, 509, 1021 or 4093, depending on the expected size of the document.

Set the state of a feature.

If a feature is not recognized by the :ref:`DOMParser <doxid-class_poco_1_1_x_m_l_1_1_d_o_m_parser>`, it is passed on to the underlying :ref:`XMLReader <doxid-class_poco_1_1_x_m_l_1_1_x_m_l_reader>`.

The only currently supported feature is `http://www.appinf.com/features/no-whitespace-in-element-content <http://www.appinf.com/features/no-whitespace-in-element-content>`__ which, when activated, causes the :ref:`WhitespaceFilter <doxid-class_poco_1_1_x_m_l_1_1_whitespace_filter>` to be used.
