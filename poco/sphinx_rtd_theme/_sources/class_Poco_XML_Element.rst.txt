.. _details-class_poco_1_1_x_m_l_1_1_element:

Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`Element <doxid-class_poco_1_1_x_m_l_1_1_element>` interface represents an element in an :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` document.

Elements may have attributes associated with them; since the :ref:`Element <doxid-class_poco_1_1_x_m_l_1_1_element>` interface inherits from :ref:`Node <doxid-class_poco_1_1_x_m_l_1_1_node>`, the generic :ref:`Node <doxid-class_poco_1_1_x_m_l_1_1_node>` interface attribute attributes may be used to retrieve the set of all attributes for an element. There are methods on the :ref:`Element <doxid-class_poco_1_1_x_m_l_1_1_element>` interface to retrieve either an :ref:`Attr <doxid-class_poco_1_1_x_m_l_1_1_attr>` object by name or an attribute value by name. In :ref:`XML <doxid-namespace_poco_1_1_x_m_l>`, where an attribute value may contain entity references, an :ref:`Attr <doxid-class_poco_1_1_x_m_l_1_1_attr>` object should be retrieved to examine the possibly fairly complex sub-tree representing the attribute value.

Puts all :ref:`Text <doxid-class_poco_1_1_x_m_l_1_1_text>` nodes in the full depth of the sub-tree underneath this :ref:`Element <doxid-class_poco_1_1_x_m_l_1_1_element>`, including attribute nodes, into a "normal" form where only markup (e.g., tags, comments, processing instructions, CDATA sections, and entity references) separates :ref:`Text <doxid-class_poco_1_1_x_m_l_1_1_text>` nodes, i.e., there are no adjacent :ref:`Text <doxid-class_poco_1_1_x_m_l_1_1_text>` nodes.
