Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A hashtable that stores :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` names consisting of an :ref:`URI <doxid-class_poco_1_1_u_r_i>`, a local name and a qualified name.

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	NamePool(unsigned long size = :ref:`POCO_XML_NAMEPOOL_DEFAULT_SIZE<doxid-_name_pool_8h_1a516badd4c3172ede09868dc7638adddd>`)

Creates a name pool with room for up to size strings.

The given size should be a suitable prime number, e.g. 251, 509, 1021 or 4093.
