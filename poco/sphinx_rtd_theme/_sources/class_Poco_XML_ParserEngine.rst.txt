.. _details-class_poco_1_1_x_m_l_1_1_parser_engine:

Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class provides an object-oriented, stream-based, low-level interface to the :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` Parser Toolkit (expat).

It is strongly recommended, that you use the SAX parser classes (which are based on this class) instead of this class, since they provide a standardized, higher-level interface to the parser.

.. ref-code-block:: cpp
	:class: doxyrest-title-code-block

	void
	setEnablePartialReads(bool flag = true)

Enable or disable partial reads from the input source.

This is useful for parsing :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` from a socket stream for a protocol like XMPP, where basically single elements are read one at a time from the input source's stream, and following elements depend upon responses sent back to the peer.

Normally, the parser always reads blocks of PARSE_BUFFER_SIZE at a time, and blocks until a complete block has been read (or the end of the stream has been reached). This allows for efficient parsing of "complete" :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` documents, but fails in a case such as XMPP, where only :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` fragments are sent at a time.
