Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class provides static methods that are UTF-8 capable variants of the same functions in ``Poco/String.h``.

The various variants of :ref:`icompare() <doxid-struct_poco_1_1_u_t_f8_1a0952766010bae80e300b992c07cbb77f>` provide case insensitive comparison for UTF-8 encoded strings.

:ref:`toUpper() <doxid-struct_poco_1_1_u_t_f8_1a21a7f2f58b199f9ed44c1664ac7f4673>`, :ref:`toUpperInPlace() <doxid-struct_poco_1_1_u_t_f8_1a89268fb9545d1710cbb67de80e19a228>`, :ref:`toLower() <doxid-struct_poco_1_1_u_t_f8_1a6f449cca10275f38a4d6c525ac05ed59>` and :ref:`toLowerInPlace() <doxid-struct_poco_1_1_u_t_f8_1a45446e355d335b8a1bd74e9d9ffb2967>` provide Unicode-based character case transformation for UTF-8 encoded strings.

:ref:`removeBOM() <doxid-struct_poco_1_1_u_t_f8_1a3d8dfc1634a053b1a3f5aeadb3bf0312>` removes the UTF-8 Byte Order Mark sequence (0xEF, 0xBB, 0xBF) from the beginning of the given string, if it's there.
