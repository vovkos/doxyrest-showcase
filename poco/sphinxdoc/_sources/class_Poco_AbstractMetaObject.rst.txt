Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A :ref:`MetaObject <doxid-class_poco_1_1_meta_object>` stores some information about a C++ class.

The :ref:`MetaObject <doxid-class_poco_1_1_meta_object>` class is used by the :ref:`Manifest <doxid-class_poco_1_1_manifest>` class. :ref:`AbstractMetaObject <doxid-class_poco_1_1_abstract_meta_object>` is a common base class for all :ref:`MetaObject <doxid-class_poco_1_1_meta_object>` in a rooted class hierarchy. A :ref:`MetaObject <doxid-class_poco_1_1_meta_object>` can also be used as an object factory for its class.
