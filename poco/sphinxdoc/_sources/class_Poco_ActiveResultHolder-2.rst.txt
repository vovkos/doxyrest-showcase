Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class holds the result of an asynchronous method invocation.

It is used to pass the result from the execution thread back to the invocation thread. The class uses reference counting for memory management. Do not use this class directly, use :ref:`ActiveResult <doxid-class_poco_1_1_active_result>` instead.
