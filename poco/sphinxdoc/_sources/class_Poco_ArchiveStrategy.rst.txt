Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`ArchiveStrategy <doxid-class_poco_1_1_archive_strategy>` is used by :ref:`FileChannel <doxid-class_poco_1_1_file_channel>` to rename a rotated log file for archiving.

Archived files can be automatically compressed, using the gzip file format.
