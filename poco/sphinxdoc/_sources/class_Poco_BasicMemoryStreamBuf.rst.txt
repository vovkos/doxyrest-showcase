Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

:ref:`BasicMemoryStreamBuf <doxid-class_poco_1_1_basic_memory_stream_buf>` is a simple implementation of a stream buffer for reading and writing from a memory area.

This streambuf only supports unidirectional streams. In other words, the :ref:`BasicMemoryStreamBuf <doxid-class_poco_1_1_basic_memory_stream_buf>` can be used for the implementation of an istream or an ostream, but not for an iostream.
