Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class provides some static methods that are used by the :ref:`poco_assert_dbg() <doxid-_bugcheck_8h_1ac13d4d417304e09659a50f39f4d7920f>`, :ref:`poco_assert() <doxid-_bugcheck_8h_1a6e02e3964cf3e2bcc686e7a8014bd6a0>`, :ref:`poco_check_ptr() <doxid-_bugcheck_8h_1a2379a6a06279aa3a2f328b7c3d646889>`, :ref:`poco_bugcheck() <doxid-_bugcheck_8h_1a01b1dc829ec1a2b5cfe35e98ce0835fe>` and :ref:`poco_unexpected() <doxid-_bugcheck_8h_1a375ce587c8acd55cea029571ff8ff76d>` macros.

You should not invoke these methods directly. Use the macros instead, as they automatically provide useful context information.
