Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`DigestInputStream <doxid-class_poco_1_1_digest_input_stream>` and :ref:`DigestOutputStream <doxid-class_poco_1_1_digest_output_stream>`.

This class is needed to ensure the correct initialization order of the stream buffer and base classes.
