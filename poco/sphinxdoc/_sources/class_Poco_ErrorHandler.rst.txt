Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This is the base class for thread error handlers.

An unhandled exception that causes a thread to terminate is usually silently ignored, since the class library cannot do anything meaningful about it.

The :ref:`Thread <doxid-class_poco_1_1_thread>` class provides the possibility to register a global :ref:`ErrorHandler <doxid-class_poco_1_1_error_handler>` that is invoked whenever a thread has been terminated by an unhandled exception. The :ref:`ErrorHandler <doxid-class_poco_1_1_error_handler>` must be derived from this class and can provide implementations of all three :ref:`exception() <doxid-class_poco_1_1_error_handler_1aa9003522d497c9e1a0d04c26cbd02fa3>` overloads.

The :ref:`ErrorHandler <doxid-class_poco_1_1_error_handler>` is always invoked within the context of the offending thread.
