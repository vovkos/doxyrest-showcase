Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The purpose of the :ref:`EventArgs <doxid-class_poco_1_1_event_args>` class is to be used as parameter when one doesn't want to send any data.

One can use :ref:`EventArgs <doxid-class_poco_1_1_event_args>` as a base class for one's own event arguments but with the arguments being a template parameter this is not necessary.
