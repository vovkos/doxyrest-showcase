	std::istream*
	open(:ref:`const<doxid-zconf_8h_1a2c212835823e3c54a8ab6d95c652660e>` :ref:`URI<doxid-class_poco_1_1_u_r_i>`& uri)

Creates and opens a file stream in binary mode for the given :ref:`URI <doxid-class_poco_1_1_u_r_i>`.

The :ref:`URI <doxid-class_poco_1_1_u_r_i>` must be either a file :ref:`URI <doxid-class_poco_1_1_u_r_i>` or a relative :ref:`URI <doxid-class_poco_1_1_u_r_i>` reference containing a path to a local file.

Throws an FileNotFound exception if the file cannot be opened.
