Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class implements a map using a :ref:`LinearHashTable <doxid-class_poco_1_1_linear_hash_table>`.

A :ref:`HashMap <doxid-class_poco_1_1_hash_map>` can be used just like a std::map.
