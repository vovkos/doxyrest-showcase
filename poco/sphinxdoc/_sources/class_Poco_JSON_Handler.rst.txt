Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Interface for handling parsing events generated by the :ref:`JSON <doxid-namespace_poco_1_1_j_s_o_n>` :ref:`Parser <doxid-class_poco_1_1_j_s_o_n_1_1_parser>`.

An application can implement a subclass of :ref:`Handler <doxid-class_poco_1_1_j_s_o_n_1_1_handler>` to implement callback-based parsing of a :ref:`JSON <doxid-namespace_poco_1_1_j_s_o_n>` document, similar to how a SAX parser would handle :ref:`XML <doxid-namespace_poco_1_1_x_m_l>`.
