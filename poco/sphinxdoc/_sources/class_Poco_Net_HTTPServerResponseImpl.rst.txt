Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This subclass of :ref:`HTTPServerResponse <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_response>` is used for representing server-side HTTP responses.

A :ref:`HTTPServerResponse <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_response>` is passed to the handleRequest() method of :ref:`HTTPRequestHandler <doxid-class_poco_1_1_net_1_1_h_t_t_p_request_handler>`.

handleRequest() must set a status code and optional reason phrase, set headers as necessary, and provide a message body.

Sends the response header to the client, followed by the contents of the given buffer.

The Content-Length header of the response is set to length and chunked transfer encoding is disabled.

If both the HTTP message header and body (from the given buffer) fit into one single network packet, the complete response can be sent in one network packet.

Must not be called after :ref:`send() <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_response_impl_1a67829f14aeb77c5b0e5212ccc6cdf982>`, :ref:`sendFile() <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_response_impl_1a2226e64760620ed11a6bb36326236bdb>` or :ref:`redirect() <doxid-class_poco_1_1_net_1_1_h_t_t_p_server_response_impl_1a778d6aadc029b00ff306507ecd1eb284>` has been called.
