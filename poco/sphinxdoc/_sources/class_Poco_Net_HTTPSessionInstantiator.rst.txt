Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A factory for :ref:`HTTPClientSession <doxid-class_poco_1_1_net_1_1_h_t_t_p_client_session>` objects.

Creates a HTTP session for a given :ref:`URI <doxid-class_poco_1_1_u_r_i>`. A :ref:`HTTPSessionInstantiator <doxid-class_poco_1_1_net_1_1_h_t_t_p_session_instantiator>` is not used directly. Instances are registered with a :ref:`HTTPSessionFactory <doxid-class_poco_1_1_net_1_1_h_t_t_p_session_factory>`, and used through it.
