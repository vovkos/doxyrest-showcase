Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for :ref:`MailInputStream <doxid-class_poco_1_1_net_1_1_mail_input_stream>` and :ref:`MailOutputStream <doxid-class_poco_1_1_net_1_1_mail_output_stream>`.

This class provides common methods and is also needed to ensure the correct initialization order of the stream buffer and base classes.
