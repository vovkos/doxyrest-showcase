Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

An input stream for reading from a socket.

When using formatted input from a :ref:`SocketInputStream <doxid-class_poco_1_1_net_1_1_socket_input_stream>`, always ensure that a receive timeout is set for the socket. Otherwise your program might unexpectedly hang.

However, using formatted input from a :ref:`SocketInputStream <doxid-class_poco_1_1_net_1_1_socket_input_stream>` is not recommended, due to the read-ahead behavior of istream with formatted reads.
