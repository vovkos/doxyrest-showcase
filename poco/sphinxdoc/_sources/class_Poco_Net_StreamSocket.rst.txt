Detailed Documentation

------------

	StreamSocket()

Creates an unconnected stream socket.

Before sending or receiving data, the socket must be connected with a call to :ref:`connect() <doxid-class_poco_1_1_net_1_1_stream_socket_1a4f7be10a71f621f3e254a0b851d34c01>`.

	StreamSocket(:ref:`IPAddress::Family<doxid-class_poco_1_1_net_1_1_i_p_address_1a048c8fc29019f4722f2cdb5ba752d9b6>` family)

Creates an unconnected stream socket for the given address family.

This is useful if certain socket options (like send and receive buffer) sizes, that must be set before connecting the socket, will be set later on.

Creates the :ref:`Socket <doxid-class_poco_1_1_net_1_1_socket>` and attaches the given :ref:`SocketImpl <doxid-class_poco_1_1_net_1_1_socket_impl>`.

The socket takes owership of the :ref:`SocketImpl <doxid-class_poco_1_1_net_1_1_socket_impl>`.

The :ref:`SocketImpl <doxid-class_poco_1_1_net_1_1_socket_impl>` must be a :ref:`StreamSocketImpl <doxid-class_poco_1_1_net_1_1_stream_socket_impl>`, otherwise an InvalidArgumentException will be thrown.

Up to length bytes are received.
