Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A factory for :ref:`TCPServerConnection <doxid-class_poco_1_1_net_1_1_t_c_p_server_connection>` objects.

The :ref:`TCPServer <doxid-class_poco_1_1_net_1_1_t_c_p_server>` class uses a :ref:`TCPServerConnectionFactory <doxid-class_poco_1_1_net_1_1_t_c_p_server_connection_factory>` to create a connection object for each new connection it accepts.

Subclasses must override the :ref:`createConnection() <doxid-class_poco_1_1_net_1_1_t_c_p_server_connection_factory_1a944c144f5c330e14963540274cab3b75>` method.

The :ref:`TCPServerConnectionFactoryImpl <doxid-class_poco_1_1_net_1_1_t_c_p_server_connection_factory_impl>` template class can be used to automatically instantiate a :ref:`TCPServerConnectionFactory <doxid-class_poco_1_1_net_1_1_t_c_p_server_connection_factory>` for a given subclass of :ref:`TCPServerConnection <doxid-class_poco_1_1_net_1_1_t_c_p_server_connection>`.
