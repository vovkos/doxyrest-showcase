Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The base class for all notification classes used with the :ref:`NotificationCenter <doxid-class_poco_1_1_notification_center>` and the :ref:`NotificationQueue <doxid-class_poco_1_1_notification_queue>` classes.

The :ref:`Notification <doxid-class_poco_1_1_notification>` class can be used with the :ref:`AutoPtr <doxid-class_poco_1_1_auto_ptr>` template class.
