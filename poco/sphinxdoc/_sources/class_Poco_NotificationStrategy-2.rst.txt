Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The interface that all notification strategies must implement.

Note: :ref:`Event <doxid-class_poco_1_1_event>` is based on policy-driven design, so every strategy implementation must provide all the methods from this interface (otherwise: compile errors) but does not need to inherit from :ref:`NotificationStrategy <doxid-class_poco_1_1_notification_strategy>`.
