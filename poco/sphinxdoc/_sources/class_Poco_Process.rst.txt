.. ref-code-block:: cpp

	static
	void
	requestTermination(:ref:`PID<doxid-class_poco_1_1_process_1ad401f1531e85e260c7bb75aad081de1f>` pid)

Requests termination of the process with the give PID.

On Unix platforms, this will send a SIGINT to the process and thus work with arbitrary processes.

On other platforms, a global event flag will be set. Setting the flag will cause :ref:`Util::ServerApplication::waitForTerminationRequest() <doxid-class_poco_1_1_util_1_1_server_application_1a17d66f7da578059c52e06a70031072e2>` to return. Therefore this will only work with applications based on :ref:`Util::ServerApplication <doxid-class_poco_1_1_util_1_1_server_application>`.
