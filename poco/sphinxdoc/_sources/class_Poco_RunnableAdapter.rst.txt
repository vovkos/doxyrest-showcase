Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This adapter simplifies using ordinary methods as targets for threads.

Usage:

.. ref-code-block:: cpp

	RunnableAdapter<MyClass> ra(myObject, &MyObject::doSomething));
	Thread thr;
	thr.Start(ra);

For using a freestanding or static member function as a thread target, please see the :ref:`ThreadTarget <doxid-class_poco_1_1_thread_target>` class.
