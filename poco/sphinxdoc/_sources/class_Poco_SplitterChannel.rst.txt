Sets or changes a configuration property.

Only the "channel" property is supported, which allows adding a comma-separated list of channels via the :ref:`LoggingRegistry <doxid-class_poco_1_1_logging_registry>`. The "channel" property is set-only. To simplify file-based configuration, all property names starting with "channel" are treated as "channel".
