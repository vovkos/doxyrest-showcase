Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The Slot template wraps another class so that it can be stored in a :ref:`ThreadLocalStorage <doxid-class_poco_1_1_thread_local_storage>` object.

This class is used internally, and you must not create instances of it yourself.
