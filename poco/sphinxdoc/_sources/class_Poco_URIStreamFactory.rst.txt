Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class defines the interface that all :ref:`URI <doxid-class_poco_1_1_u_r_i>` stream factories must implement.

Subclasses must implement the :ref:`open() <doxid-class_poco_1_1_u_r_i_stream_factory_1a0768071cef5d26530977d4fd23a8abc8>` method.

Tries to create and open an input stream for the resource specified by the given :ref:`URI <doxid-class_poco_1_1_u_r_i>`.

If the stream cannot be opened for whatever reason, an appropriate IOException must be thrown.

If opening the stream results in a redirect, a :ref:`URIRedirection <doxid-class_poco_1_1_u_r_i_redirection>` exception should be thrown.
