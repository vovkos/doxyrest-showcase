Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class contains enumerations and static utility functions for dealing with :ref:`Unicode <doxid-class_poco_1_1_unicode>` characters and their properties.

For more information on :ref:`Unicode <doxid-class_poco_1_1_unicode>`, see `http://www.unicode.org <http://www.unicode.org>`__.

The implementation is based on the :ref:`Unicode <doxid-class_poco_1_1_unicode>` support functions in PCRE.
