Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

This class template simplifies the implementation of :ref:`TimerTask <doxid-class_poco_1_1_util_1_1_timer_task>` objects by allowing a member function of an object to be called as task.

------------

Creates the :ref:`TimerTaskAdapter <doxid-class_poco_1_1_util_1_1_timer_task_adapter>`, using the given object and its member function as task target.

The member function must accept one argument, a reference to a :ref:`TimerTask <doxid-class_poco_1_1_util_1_1_timer_task>` object.
