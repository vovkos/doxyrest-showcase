Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A :ref:`Value <doxid-class_poco_1_1_util_1_1_units_1_1_value>` with a unit.

.. ref-code-block:: cpp

	- V is the type you are storing
	- U is the unit of the Value

This class is usually not used directly; client code should use the typedef'd instantiations defined in the :ref:`Values <doxid-namespace_poco_1_1_util_1_1_units_1_1_values>` and :ref:`Constants <doxid-namespace_poco_1_1_util_1_1_units_1_1_constants>` namespace.

Example:

.. ref-code-block:: cpp

	using namespace Poco::Util::Units::Values;
	
	std::cout << "One mile is " << km(mile(1)) << std::endl;
	    // Output: One mile is 1.60934 km
	
	std::cout << "Flow rate is " << m3(mile(1)*inch(80)*foot(9))/s(minute(5));
	    // Output: Flow rate is 29.9026 (m)^3.(s)^-1
	
	hours h;
	h = cm(3);   // Compile-time error: incompatible units
	h = 4;       // Compile-time error: 4 of what?
	h = days(4); // Ok: h is 96 hours
