Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`DocumentEvent <doxid-class_poco_1_1_x_m_l_1_1_document_event>` interface provides a mechanism by which the user can create an :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` of a type supported by the implementation.

It is expected that the :ref:`DocumentEvent <doxid-class_poco_1_1_x_m_l_1_1_document_event>` interface will be implemented on the same object which implements the :ref:`Document <doxid-class_poco_1_1_x_m_l_1_1_document>` interface in an implementation which supports the :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` model.

	createEvent(:ref:`const<doxid-zconf_8h_1a2c212835823e3c54a8ab6d95c652660e>` :ref:`XMLString<doxid-namespace_poco_1_1_x_m_l_1a288dd5005eea5f995c0169e0688593ae>`& eventType) const = 0

Creates an event of the specified type.

The eventType parameter specifies the type of :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` interface to be created. If the :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` interface specified is supported by the implementation this method will return a new :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` of the interface type requested. If the :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` is to be dispatched via the dispatchEvent method the appropriate event init method must be called after creation in order to initialize the :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` 's values. As an example, a user wishing to synthesize some kind of UIEvent would call createEvent with the parameter "UIEvents". The initUIEvent method could then be called on the newly created UIEvent to set the specific type of UIEvent to be dispatched and set its context information. The createEvent method is used in creating Events when it is either inconvenient or unnecessary for the user to create an :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` themselves. In cases where the implementation provided :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` is insufficient, users may supply their own :ref:`Event <doxid-class_poco_1_1_x_m_l_1_1_event>` implementations for use with the dispatchEvent method.
