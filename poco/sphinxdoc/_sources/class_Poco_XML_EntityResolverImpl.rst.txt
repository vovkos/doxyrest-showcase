Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

A default implementation of the :ref:`EntityResolver <doxid-class_poco_1_1_x_m_l_1_1_entity_resolver>` interface.

The system ID is first interpreted as an :ref:`URI <doxid-class_poco_1_1_u_r_i>` and the :ref:`URIStreamOpener <doxid-class_poco_1_1_u_r_i_stream_opener>` is used to create and open an istream for an :ref:`InputSource <doxid-class_poco_1_1_x_m_l_1_1_input_source>`.

If the system ID is not a valid :ref:`URI <doxid-class_poco_1_1_u_r_i>`, it is interpreted as a filesystem path and a :ref:`Poco::FileInputStream <doxid-class_poco_1_1_file_input_stream>` is opened for it.

Tries to use the :ref:`URIStreamOpener <doxid-class_poco_1_1_u_r_i_stream_opener>` to create and open an istream for the given systemId, which is interpreted as an :ref:`URI <doxid-class_poco_1_1_u_r_i>`.
