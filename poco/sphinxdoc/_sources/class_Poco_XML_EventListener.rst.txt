Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

The :ref:`EventListener <doxid-class_poco_1_1_x_m_l_1_1_event_listener>` interface is the primary method for handling events.

Users implement the :ref:`EventListener <doxid-class_poco_1_1_x_m_l_1_1_event_listener>` interface and register their listener on an :ref:`EventTarget <doxid-class_poco_1_1_x_m_l_1_1_event_target>` using the AddEventListener method. The users should also remove their :ref:`EventListener <doxid-class_poco_1_1_x_m_l_1_1_event_listener>` from its :ref:`EventTarget <doxid-class_poco_1_1_x_m_l_1_1_event_target>` after they have completed using the listener.

When a :ref:`Node <doxid-class_poco_1_1_x_m_l_1_1_node>` is copied using the cloneNode method the EventListeners attached to the source :ref:`Node <doxid-class_poco_1_1_x_m_l_1_1_node>` are not attached to the copied :ref:`Node <doxid-class_poco_1_1_x_m_l_1_1_node>`. If the user wishes the same EventListeners to be added to the newly created copy the user must add them manually.
