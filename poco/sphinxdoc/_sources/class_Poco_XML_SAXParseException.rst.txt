Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Encapsulate an :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` parse error or warning.

This exception may include information for locating the error in the original :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` document, as if it came from a :ref:`Locator <doxid-class_poco_1_1_x_m_l_1_1_locator>` object. Note that although the application will receive a :ref:`SAXParseException <doxid-class_poco_1_1_x_m_l_1_1_s_a_x_parse_exception>` as the argument to the handlers in the :ref:`ErrorHandler <doxid-class_poco_1_1_x_m_l_1_1_error_handler>` interface, the application is not actually required to throw the exception; instead, it can simply read the information in it and take a different action.

Since this exception is a subclass of SAXException, it inherits the ability to wrap another exception.
