Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Interface for an :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` filter.

An :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` filter is like an :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` reader, except that it obtains its events from another :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` reader rather than a primary source like an :ref:`XML <doxid-namespace_poco_1_1_x_m_l>` document or database. Filters can modify a stream of events as they pass on to the final application.

The :ref:`XMLFilterImpl <doxid-class_poco_1_1_x_m_l_1_1_x_m_l_filter_impl>` helper class provides a convenient base for creating SAX2 filters, by passing on all :ref:`EntityResolver <doxid-class_poco_1_1_x_m_l_1_1_entity_resolver>`, :ref:`DTDHandler <doxid-class_poco_1_1_x_m_l_1_1_d_t_d_handler>`, :ref:`ContentHandler <doxid-class_poco_1_1_x_m_l_1_1_content_handler>` and :ref:`ErrorHandler <doxid-class_poco_1_1_x_m_l_1_1_error_handler>` events automatically.
