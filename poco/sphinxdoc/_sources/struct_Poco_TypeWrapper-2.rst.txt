template struct Poco::TypeWrapper
=================================

.. toctree::
	:hidden:

Use the type wrapper if you want to decouple constness and references from template types.


.. ref-code-block:: cpp
	:class: doxyrest-overview-code-block

	#include <MetaProgramming.h>
	
	template <typename T>
	struct TypeWrapper
	{
		// typedefs
	
		typedef T :target:`TYPE<doxid-struct_poco_1_1_type_wrapper_1ac5a059b745d0a24900f175f77ef18f3f>`;

		typedef T& :target:`REFTYPE<doxid-struct_poco_1_1_type_wrapper_1a78c594345651cb155d64d156b432aa5e>`;

	};
