enum Poco::Net::RemoteSyslogChannel::Severity

Overview

#include <RemoteSyslogChannel.h>

enum Severity
{
    SYSLOG_EMERGENCY     = 0,
    SYSLOG_ALERT         = 1,
    SYSLOG_CRITICAL      = 2,
    SYSLOG_ERROR         = 3,
    SYSLOG_WARNING       = 4,
    SYSLOG_NOTICE        = 5,
    SYSLOG_INFORMATIONAL = 6,
    SYSLOG_DEBUG         = 7,
};

Detailed Documentation

Enum Values

SYSLOG_EMERGENCY

Emergency: system is unusable.

SYSLOG_ALERT

Alert: action must be taken immediately.

SYSLOG_CRITICAL

Critical: critical conditions.

SYSLOG_ERROR

Error : error conditions.

SYSLOG_WARNING

Warning: warning conditions.

SYSLOG_NOTICE

Notice: normal but significant condition.

SYSLOG_INFORMATIONAL

Informational: informational messages.

SYSLOG_DEBUG

Debug: debug-level messages.
